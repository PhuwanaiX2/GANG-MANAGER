# GangManager Logo

## Concept

Current temporary logo: `logov2`

GangManager uses a bold `G` mark for now so the brand feels more gang-first and less Discord-native while the final brand direction is still being refined.

## Assets

- Product mark: `apps/web/public/brand/logov2.svg`
- Browser icon source: `apps/web/src/app/icon.svg`
- PNG source artwork: `apps/web/public/brand/logov2.png`
- React UI component: `apps/web/src/components/BrandLogo.tsx`

## Usage

Use `BrandLogo` for product navigation and footer lockups. Use `BrandMark` where the UI only has room for an icon-sized product signal. Treat `logov2` as a temporary production placeholder.
